# Netflix Movies & TV Shows - Data Visualization

## Project Overview

This project focuses on visualizing the Netflix Movies & TV Shows dataset using Python. It presents meaningful charts and graphs to identify trends, distributions, and insights from Netflix's content library.

## Tools & Libraries

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Netflix Movies and TV Shows Dataset

## Visualizations Included

- Movies vs TV Shows
- Top 10 Countries
- Content Ratings
- Top 10 Genres
- Content Added by Year
- Top Directors
- Release Year Trend
- Movies vs TV Shows Pie Chart
